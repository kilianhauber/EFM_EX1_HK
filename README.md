# Projektübersicht

Dieses Projekt analysiert die Unterschiede im Kaufverhalten zwischen Schweizer und chinesischen Konsumenten. Es wird untersucht, wie kulturelle Unterschiede westlichen Unternehmen Herausforderungen bereiten, wenn sie versuchen, in den chinesischen Markt einzutreten. Die Analyse basiert auf Daten und Quellen, die in der Datei `quellen.json` referenziert werden.
